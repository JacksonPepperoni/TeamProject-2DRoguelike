## 🖥️ 1500000000000의 번제

+ 2D 로그라이크 "아이작의 번제" 모티브로 만든 게임
+ 이동 [W, A, S, D]
+ 공격 [←, →, ↑, ↓]
+ 폭탄 [E]
+ 일시정지 [P]

## 📆 Develop Schedule

* 23.12.15 ~ 23.12.21

## ⚙️ Environment

- `Unity 2022.3.2`
- **IDE** : Visual Studio 2022
- **VCS** : Git (GitHub Desktop)
- **Envrionment** : PC `only`
- **Resolution** :	1920 x 1080 `FHD`
